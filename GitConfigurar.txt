git add *  para agregar archivos y preparar para commit

git commit -m "COMENTARIO"  para subir al repositorio

git status   muestra el estatus del proyecto


git log   muestra el LOG de lo que se ha realizado

git push origin master  para subir a la nube y poder tenerlo en github

git revert ID  para regresar un cambio.

git branch  lista las ramas que tenemos localmente
git checkout -b nueva_rama   crear una rama
git checkout rama   para cambiarnos de rama
git branch -d rama  para borrar una branch
git branch -D rama forza el borrado de la rama **Cuidado con esta Opcion**
git merge para mesclar los cambios en cada rama
